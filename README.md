# Space One

A modern animated landing page built with **Next.js, React, Tailwind CSS and GSAP**.
The project demonstrates smooth animations, interactive UI elements and a clean responsive design.

Live Demo
https://space-one-drab.vercel.app/

---

## Overview

Space One is a visually focused frontend project created to showcase modern web UI techniques including animation, smooth transitions and responsive layouts.

The application focuses on user experience, fluid motion and elegant design.

---

## Features

* Smooth animations powered by **GSAP**
* Modern UI built with **React and Next.js**
* Responsive layout for desktop and mobile devices
* Tailwind CSS utility-first styling
* Clean component-based architecture
* Optimized rendering with Next.js

---

## Installation

Clone the repository

```bash
git clone https://github.com/romatimanov/your-repository-name.git
```

Go to project directory

```bash
cd your-repository-name
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
